# 📖 Module Overview

This module covers the core building blocks of the C# programming language. Before learning Object-Oriented Programming (OOP), it is essential to understand variables, data types, operators, arrays, loops, methods, and type conversion.

These concepts are the foundation of every C# application, whether you are building a Console Application, Windows Forms application, ASP.NET MVC application, or Web API.

---

# 🎯 Learning Objectives

After completing this module, you will be able to:

- Declare and use variables.
- Understand different C# data types.
- Follow proper variable naming conventions.
- Perform implicit and explicit type conversion.
- Use arithmetic, relational, logical, and assignment operators.
- Create and manipulate arrays.
- Work with multidimensional arrays.
- Use loops to repeat tasks.
- Write reusable methods.
