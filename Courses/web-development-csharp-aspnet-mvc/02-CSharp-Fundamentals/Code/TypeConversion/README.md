# Type Conversion

This folder contains examples demonstrating different types of type conversion in C#.

## Contents

- Implicit Conversion
- Explicit Conversion
- Convert Class
- Parse vs TryParse
- Boxing and Unboxing
- Practice Exercises

---

## Learning Objectives

After completing this folder, you will be able to:

- Understand implicit and explicit conversion.
- Convert between different data types.
- Use Convert class effectively.
- Understand Parse() and TryParse().
- Explain Boxing and Unboxing.
- Write safe conversion code.

---

| File | Description |
|------|-------------|
| Program.cs | Entry point |
