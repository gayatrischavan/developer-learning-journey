# Naming Rules - Code Examples

This folder contains practical examples demonstrating **Naming Rules and Naming Conventions** in C#.

---

## Files

| File | Description |
|------|-------------|
| Program.cs | Main program demonstrating naming rules |
| ValidIdentifiers.cs | Examples of valid identifiers |
| InvalidIdentifiers.cs | Examples of invalid identifiers (commented) |
| CamelCasePascalCase.cs | Naming convention examples |
| PracticeExercises.cs | Practice programs |

## Topics Covered

- Identifiers
- Variable Naming Rules
- Reserved Keywords
- camelCase
- PascalCase
- Best Practices

---

## Learning Outcome

After completing these examples, you will be able to:

- Create valid identifiers
- Avoid invalid variable names
- Follow Microsoft's naming conventions
- Write clean and readable code
