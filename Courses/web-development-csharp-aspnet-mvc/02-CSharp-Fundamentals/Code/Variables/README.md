# Variables - Code Examples

This folder contains practical examples related to **Variables in C#**.

## Files

| File                      | Description                                 |
| ------------------------- | ------------------------------------------- |
| Program.cs                | Demonstrates all variable concepts together |
| VariableDeclaration.cs    | Variable declaration examples               |
| VariableInitialization.cs | Variable initialization examples            |
| VariableScope.cs          | Local variable scope examples               |
| Constants.cs              | Constant variable examples                  |

---

## Prerequisites

- Visual Studio
- .NET SDK
- Basic understanding of Console Applications

---

## Topics Covered

- Variable Declaration
- Variable Initialization
- Variable Assignment
- Variable Scope
- Constants (`const`)
- Best Practices

---

## Learning Outcome

After completing these examples, you will be able to:

- Declare variables
- Initialize variables
- Update variable values
- Understand variable scope
- Use constants correctly
