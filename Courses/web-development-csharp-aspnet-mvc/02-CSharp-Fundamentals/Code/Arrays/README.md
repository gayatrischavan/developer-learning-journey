# Arrays - Code Examples

This folder contains practical examples demonstrating C# Arrays.

|------|-------------|
| Program.cs | Entry point |
| ArrayDeclaration.cs | Declaring arrays |
| ArrayInitialization.cs | Initializing arrays |

## Learning Objectives

After completing these examples, you will learn:

- Declare arrays
- Initialize arrays
- Access array elements
- Update array values
- Find array length
- Understand array indexing
