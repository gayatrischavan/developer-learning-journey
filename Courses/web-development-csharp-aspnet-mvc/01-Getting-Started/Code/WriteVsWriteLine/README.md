# Console.Write() vs Console.WriteLine()

This folder contains examples demonstrating the difference between `Console.Write()` and `Console.WriteLine()`.

## Files

| File                 | Description                      |
| -------------------- | -------------------------------- |
| Program.cs           | Main demonstration               |
| WriteExample.cs      | Demonstrates Console.Write()     |
| WriteLineExample.cs  | Demonstrates Console.WriteLine() |
| ComparisonExample.cs | Side-by-side comparison          |

## Learning Outcome

After completing these examples, you will understand:

- How `Write()` works
- How `WriteLine()` works
- When to use each method
- Best practices for console output
