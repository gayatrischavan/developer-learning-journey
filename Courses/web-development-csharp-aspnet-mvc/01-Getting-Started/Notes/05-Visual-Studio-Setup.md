# Visual Studio Setup

## Overview

Visual Studio is Microsoft's Integrated Development Environment (IDE) used to build C#, .NET, ASP.NET, and many other applications.

---

## Why do we need Visual Studio?

It provides everything needed to develop applications in one place.

Features include:

- Code Editor
- IntelliSense
- Debugger
- Designer
- Compiler
- NuGet Package Manager
- Git Integration

---

## Installation Steps

1. Download Visual Studio.
2. Run the installer.
3. Select the **.NET Desktop Development** workload.
4. Click Install.
5. Launch Visual Studio.

---

## Creating Your First Console Application

1. Open Visual Studio.
2. Click **Create a new project**.
3. Select **Console App (.NET)**.
4. Click Next.
5. Enter Project Name.
6. Click Create.

## Best Practices

✔ Use meaningful project names.

✔ Save projects inside a dedicated Learning folder.

✔ Learn keyboard shortcuts.

✔ Build frequently.

## Common Shortcuts

| Shortcut         | Action                |
| ---------------- | --------------------- |
| Ctrl + Shift + B | Build                 |
| F5               | Run                   |
| Ctrl + F5        | Run without Debugging |
| Ctrl + K + C     | Comment               |
| Ctrl + K + U     | Uncomment             |
| Ctrl + .         | Quick Fix             |
