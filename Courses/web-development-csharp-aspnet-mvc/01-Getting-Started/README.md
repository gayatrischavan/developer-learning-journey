# 📘 Module 01 - Getting Started with C#

> **Course:** Web Development with C#, ASP.NET MVC, MS SQL & C# OOP  
> **Module:** 01 - Getting Started

---

# 📖 Module Overview

This module introduces the fundamentals of C# programming and the .NET development environment.

You will learn how to create your first Console Application, use the `Console` class for input and output, understand how a C# program executes, and set up Visual Studio for development.

This module serves as the foundation for all upcoming C# topics.

---

# 🎯 Learning Objectives

After completing this module, you will be able to:

- Understand what a Console Application is.
- Create and run your first C# project.
- Use the Console class for input and output.
- Understand the execution flow of a C# program.
- Explain the roles of CLR, IL, and JIT Compiler.
- Compare Console Applications with Windows Forms.
- Use Visual Studio efficiently.
